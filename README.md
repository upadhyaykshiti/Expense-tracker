Expense Tracker (Backend + Frontend)
-----------------------------------
Backend: Node.js + Express + MongoDB (in /backend)
Frontend: React (Vite) + Redux Toolkit + Tailwind + Recharts (in /frontend)

Quick start:
1. Clone the Repository:
   - git clone https://github.com/upadhyaykshiti/Expense-tracker.git
   - cd Expense-tracker

2. Backend:
   - cd backend
   - copy .env.example to .env 
   - npm install
   - npm run dev

3. Frontend:
   - cd frontend
   - npm install
   - npm run dev
